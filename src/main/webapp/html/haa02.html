공판장별 품목경락가격

  • ◎ 이용안내
  • 경매 속수량이 많은 상위 6개 품목에 대한 경락가격을 공판장별로 비교분석 할수 있는 화면
  • 최근1개월, 최근6개월, 최근1년 버튼 선택 시, 선택한 날짜의 데이터가 그래프에 표출
  • 조회목록의 1순위의 품목명이 그래프의 1순위의 품목명입니다. (품목명 : 1순위=1순위, 2순위=2순위 ....)
  • 자료출처 : 자료전송을 통한 각 공판장별 경매내역
